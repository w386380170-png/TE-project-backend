# 项目需求文档

## 一、项目目标

构建一个爬虫数据可视化平台后端，满足以下功能：

- 爬取百度实时热搜 Top10 数据并入库
- 爬取豆瓣 Top100 电影数据并入库
- 提供前端查询接口，返回前端可视化展示所需数据格式
- 为后续 React + Ant Design 前端提供菜单驱动的图表切换接口
- 最终支持登录/注册模块和手写数字识别上传图片计算模块

## 二、周任务规划

### 第一周

1. 安装配置 MySQL 数据库
2. 复习 MySQL 数据库、表、记录的增删改查语句
3. 使用命令行创建学生表，包含字段：学生、姓名、身高
4. 开发 `MySQLHelper` 类，封装 MySQL 操作，并考虑方法参数设计

### 第二周

1. 学习 Python 爬虫基础知识
2. 爬取百度热搜 top10 并使用本周开发的类存入数据库
3. 深入理解爬虫原理

### 第三周

1. 爬取豆瓣电影 top100 并存入数据库
2. 根据可视化服务展示需要设计数据表字段

### 第四周

1. 学习 React（create-react-app）和 Flask/后端联调
2. 前端设计三个输入框和两个按钮：第一个按钮点击后把第一个输入框内容作为参数发送给后端，后端返回字符串
3. 第二、第三个输入框作为 POST 请求的 body 和 param，后端返回 body 和 param 中的参数值

### 第五周

1. 前端引入 echarts for react 做数据库可视化展示
2. 后端接口查询数据库，返回给前端所需的数据格式
3. 引入 ant design for react，做一个菜单，根据点击菜单展示不同图表

### 第六周

1. 完成数据可视化平台的登录注册模块

### 第七、八周

1. 阅读指定书籍，手动用 Python 在本地实现神经网络手写数字识别，不能使用第三方包

### 第九周

1. 将手写数字识别接入数据可视化平台，前端上传一张图片，后端计算并返回结果

## 三、当前后端实现现状

### 技术栈

- FastAPI
- pymysql
- MySQL
- requests
- BeautifulSoup

### 现有 API

#### 根接口

- `GET /`
  - 返回服务运行状态和文档地址

#### 百度热搜

- `GET /api/baidu/hot/crawl`
  - 触发爬取百度热搜 Top10 并入库
  - 返回爬取数量和样例数据

- `GET /api/baidu/hot/list`
  - 查询已存储的百度热搜数据
  - 返回 `{ code: 200, data: [...] }`

#### 豆瓣电影

- `GET /api/douban/movie/crawl`
  - 触发爬取豆瓣 Top100 并入库
  - 返回爬取总数

- `GET /api/douban/movie/list`
  - 查询已存储豆瓣电影明细
  - 返回前 100 条电影记录

- `GET /api/douban/stat/score`
  - 查询电影评分区间统计
  - 返回按评分区间分组的电影数量

- `GET /api/douban/stat/country`
  - 查询电影国家统计
  - 返回按国家分组的电影数量

### 数据库表设计

#### baidu_hot

- `id` INT AUTO_INCREMENT PRIMARY KEY
- `hot_title` VARCHAR(512) NOT NULL
- `hot_score` VARCHAR(128)
- `hot_url` VARCHAR(1024)
- `create_time` TIMESTAMP DEFAULT CURRENT_TIMESTAMP

#### douban_movies

- `id` INT AUTO_INCREMENT PRIMARY KEY
- `movie_name` VARCHAR(512) NOT NULL
- `score` DECIMAL(3,1)
- `director` VARCHAR(256)
- `actors` TEXT
- `release_year` INT
- `movie_type` VARCHAR(256)
- `country` VARCHAR(128)
- `summary` TEXT
- `movie_url` VARCHAR(1024)
- `crawl_time` TIMESTAMP DEFAULT CURRENT_TIMESTAMP

## 四、前端可视化接口建议

### 百度热搜图表

- `GET /api/baidu/hot/list`
  - 返回热搜标题、热度、链接
  - 可用于柱状图、折线图、表格展示

### 豆瓣电影图表

- `GET /api/douban/movie/list`
  - 电影列表数据，可用于表格展示与详细页面

- `GET /api/douban/stat/score`
  - 评分分布，用于饼图/柱状图

- `GET /api/douban/stat/country`
  - 国家分布，用于饼图/雷达图
